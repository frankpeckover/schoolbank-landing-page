# MeritBank Landing Page

A small static landing/info page for the MeritBank school wallet and reward economy app.

## Files

- `index.html` contains the page content and structure.
- `styles.css` contains all visual styling and responsive layout rules.
- `assets/screenshot-sign-in.png`, `assets/screenshot-dashboard.png`, and `assets/screenshot-users.png` are screenshots from the app used by the page.
- `assets/meritbank-logo.svg` is copied from the app and used in the header and footer.

## Replacing Screenshots

Export your new screenshots as PNG files and replace these files with the same names:

- `assets/screenshot-sign-in.png` for the hero sign-in image.
- `assets/screenshot-dashboard.png` for the hero overlay and analytics section.
- `assets/screenshot-users.png` for the users section.

No HTML changes are needed if the filenames stay the same.

## Local Preview

Open `index.html` directly in a browser. No build step is required.
